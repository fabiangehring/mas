Kapitel 2 Daten

Für die Analyse die Aktien grosser Unternehmen weltweit herangezogen - konkret alle Aktienkomponenten des “iShares MSCI World UCITS ETF” (vgl. Blackrock 2020) per 28. Februar 20201 verwendet. Insgesamt enthält dieser ETF 1639 Titel. Für diese stehen verschiedene Attribute zur Verfügung, welche für die Aktie von “Nestlé” wie folgt lauten:

Für all diese Titel werden - sofern verfügbar - werden die täglichen historischen Aktienkurse inkl. Höchst- und Tiefstkurs bezogen. Diese Daten sind via Yahoo Finance frei verfügbar (vgl. Yahoo 2020). Für die Analyse kommen so 9’212’052 tägliche Datenwerte für 1’636 Unternehmen zusammen. Für 3 Titel können keine Werte gefunden werden. Tabelle 2.2 zeigt einen Beispieleintrag für die Aktie von Nesté per 14. Februar 2019. Dieser Tag ist insofern speziell als dass es der letzte Börsentag vor Ex-Dividend, d.h. der letzte Tag, bevor die Aktie ohne die für das Jahr 2019 ausgeschüttete Dividende gahandelt wird. Diese betrug CHF 2.45 (Nestle 2020) und wiederspiegelt sich in den Daten als Differenz des Close- und Adjusted-Preises. Zu beachten gilt es ferner, dass die Yahoo Ticker für ausserhalb der USA gehandelte Titel einen Suffix für das Länderkürzel haben.
